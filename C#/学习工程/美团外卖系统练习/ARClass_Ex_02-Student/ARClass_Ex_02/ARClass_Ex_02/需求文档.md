# 美团外卖系统需求文档

## 1. 系统概述

美团外卖系统是一个控制台应用程序，模拟了外卖平台的完整流程。系统通过面向对象的设计，实现了顾客下单、快递员接单和配送的完整流程。

## 2. 功能需求

### 2.1 顾客功能
- **下单**：选择商家、选择商品，系统自动使用顾客注册时的默认地址进行下单

### 2.2 快递员功能
- **自动接单**：下单成功后5秒自动分配空闲快递员接单
- **自动配送**：接单后10秒自动完成配送并通知顾客取餐

### 2.3 顾客操作界面
顾客登录后可以进行以下操作：
1. 下单
2. 返回上一级

## 3. 系统架构

### 3.1 核心类
- **Order**：订单类，管理订单信息和状态
- **Customer**：顾客类，处理顾客相关操作
- **Merchant**：商家类，处理商家相关操作
- **Deliveryman**：快递员类，处理配送相关操作
- **Meituan**：美团平台类，作为中间协调者
- **OrderItem**：订单项类，代表订单中的单个商品
- **MenuItem**：菜单项类，代表商家菜单中的单个菜品

### 3.2 订单状态
- **Pending**：待处理（订单刚创建时的状态）
- **DeliveryAccepted**：快递员已接单
- **Delivered**：已送达
- **Completed**：已完成
- **Cancelled**：已取消

## 4. 业务流程

1. **顾客下单流程**：
   - 顾客选择商家
   - 查看商家菜单
   - 选择商品和数量
   - 系统自动使用顾客的默认地址
   - 系统生成订单（状态：Pending）

2. **快递员接单流程**：
   - 等待5秒
   - 系统自动分配空闲快递员
   - 快递员接单（状态：DeliveryAccepted）

3. **配送流程**：
   - 等待10秒配送时间
   - 快递员送达（状态：Delivered）
   - 通知顾客取餐

## 5. 技术实现

- **开发语言**：C#
- **框架**：.NET Framework 4.7.2
- **运行环境**：Windows 操作系统
- **用户界面**：控制台命令行界面

## 7. 总结

美团外卖系统实现了完整的外卖流程，包括顾客下单、快递员自动接单和配送。顾客可以选择商家和商品进行下单，系统会自动使用顾客注册时填写的默认地址作为配送地址，并自动分配快递员完成配送。系统设计简洁，流程清晰。
