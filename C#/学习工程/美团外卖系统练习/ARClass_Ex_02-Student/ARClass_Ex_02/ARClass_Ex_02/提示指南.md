# C# OOP 填空练习 — 提示指南

本指南提供关键知识点的提示，帮助完成填空练习。

---

## 1. List<T> 常用操作

### 添加元素

```csharp
List<string> names = new List<string>();
names.Add("张三");
```

### 遍历元素

```csharp
foreach (string name in names)
{
    Console.WriteLine(name);
}
```

### 按索引访问

```csharp
string first = names[0];
```

### LINQ 查询（需要 `using System.Linq;`）

```csharp
// 查找第一个满足条件的元素，找不到返回 null（引用类型）
Person found = people.FirstOrDefault(p => p.Age > 18);

// 查找第一个满足条件的元素
Product product = products.FirstOrDefault(p => p.Name == "苹果");
```

## 2. 枚举（enum）的使用

### 定义

```csharp
public enum OrderStatus
{
    Pending,
    Delivered,
    Cancelled
}
```

### 赋值与比较

```csharp
OrderStatus status = OrderStatus.Pending;

// 赋值
status = OrderStatus.Delivered;

// 比较
if (status == OrderStatus.Delivered)
{
    Console.WriteLine("已送达");
}
```

## 3. this 关键字

在实例方法中，`this` 代表当前对象实例。

```csharp
public class Customer
{
    public string Name { get; private set; }

    public Order PlaceOrder(Merchant merchant, string address, List<OrderItem> items)
    {
        // 这里 this 代表当前 Customer 对象
        Order order = new Order(this, merchant, address);
        // ...
    }
}
```

## 5. 计算属性（Expression-bodied Property）

```csharp
public decimal TotalPrice => Price * Quantity;
```

等价于：

```csharp
public decimal TotalPrice
{
    get { return Price * Quantity; }
}
```

## 6. Dictionary<K, V> 基本操作

### 创建

```csharp
Dictionary<string, decimal> prices = new Dictionary<string, decimal>();
```

### 添加

```csharp
prices.Add("苹果", 5.5m);
prices["香蕉"] = 3.0m;
```

### 访问

```csharp
decimal applePrice = prices["苹果"];
```

### 检查是否存在

```csharp
if (prices.ContainsKey("苹果"))
{
    Console.WriteLine(prices["苹果"]);
}
```

### 遍历

```csharp
foreach (var pair in prices)
{
    Console.WriteLine($"{pair.Key}: {pair.Value}元");
}
```
